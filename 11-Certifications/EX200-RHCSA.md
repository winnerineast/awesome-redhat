# EX200 — RHCSA
> Red Hat Certified System Administrator — the foundation certification.

## Exam Objectives
- Essential file management, users, groups, permissions
- SELinux configuration and troubleshooting
- Storage management (LVM, Stratis)
- Network configuration
- Service management (systemd)
- Container basics (Podman)

## Preparation
- Courses: RH124 + RH134, or RH199 (rapid track)
