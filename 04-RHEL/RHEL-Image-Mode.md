# RHEL Image Mode
> RHEL 10+ treats the OS like a container image for standardized deployment.

## Key Concepts
- Bootable container images
- GitOps workflows for OS updates
- Immutable infrastructure pattern
- Similar to CoreOS but for general RHEL workloads
