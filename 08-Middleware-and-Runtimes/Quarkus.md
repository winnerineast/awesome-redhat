# Quarkus
> Supersonic Subatomic Java — Kubernetes-native framework optimized for GraalVM and OpenJDK HotSpot.

## Key Features
- Fast startup (~0.015s with native), low memory (~12MB RSS)
- Live coding with dev mode (`quarkus dev`)
- Unified imperative and reactive programming
- Extensions ecosystem (500+ extensions)
- Native image compilation with GraalVM
- Dev Services (auto-provisioned databases, Kafka, etc.)
