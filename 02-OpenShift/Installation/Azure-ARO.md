# Azure Red Hat OpenShift (ARO)
> ARO is a fully managed OpenShift service jointly operated by Microsoft and Red Hat on Azure.

## Key Features
- Jointly managed control plane by Microsoft and Red Hat
- Integrated with Azure AD for identity
- Azure Monitor and Log Analytics integration
- SLA-backed availability
