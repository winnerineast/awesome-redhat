# ROSA — Red Hat OpenShift Service on AWS
> ROSA is a fully managed OpenShift service on AWS, jointly supported by Red Hat and Amazon.

## Key Features
- Native AWS integration (VPC, IAM, CloudWatch)
- Pay-as-you-go or annual billing through AWS
- 99.95% SLA
- HCP (Hosted Control Planes) mode for reduced overhead
