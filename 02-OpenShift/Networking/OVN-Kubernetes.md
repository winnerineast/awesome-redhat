# OVN-Kubernetes
> OVN-Kubernetes is the default CNI network provider for OpenShift 4.x, based on Open Virtual Network.

## Key Features
- NetworkPolicy support
- Egress IP / Egress Firewall
- IPsec encryption between nodes
- Hybrid networking (Linux + Windows nodes)
- IPv4, IPv6, and dual-stack support
