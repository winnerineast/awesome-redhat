# Image Streams
> ImageStreams provide an abstraction layer for container image references, enabling automatic deployment triggers.

## Key Features
- Track multiple tags/versions of an image
- Trigger rebuilds/redeployments when base images change
- Abstract registry locations (portable across environments)
- Scheduled import from external registries
