# OpenShift Data Foundation (ODF)
> ODF (formerly OCS) provides software-defined storage for OpenShift based on Ceph and Noobaa.

## Features
- Block storage (RBD) — RWO
- File storage (CephFS) — RWX
- Object storage (Noobaa/MCG) — S3-compatible
- Disaster recovery and data replication
- Encryption at rest
