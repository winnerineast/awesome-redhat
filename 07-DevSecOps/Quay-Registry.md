# Quay Registry
> Red Hat's enterprise container registry with security scanning, image signing, and geo-replication.

## Features
- Image vulnerability scanning (Clair)
- Robot accounts for CI/CD
- Repository mirroring
- Geo-replication
- Image signing and verification
- Quay.io (SaaS) or self-hosted
