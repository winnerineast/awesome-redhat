# GPU and NVIDIA Integration
> Running GPU workloads on OpenShift with the NVIDIA GPU Operator.

## Components
- **GPU Operator** — Manages NVIDIA drivers, container toolkit, device plugin
- **Node Feature Discovery** — Auto-detects GPU hardware
- **GPU scheduling** — Request GPUs via resource limits
- **Multi-Instance GPU (MIG)** — Split a single GPU into multiple instances
