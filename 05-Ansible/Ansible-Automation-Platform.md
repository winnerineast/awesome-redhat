# Ansible Automation Platform
> Enterprise automation with Controller, Hub, and Execution Environments.

## Components
- **Automation Controller** (formerly Tower) — Web UI, RBAC, workflows, scheduling
- **Automation Hub** — Curated content collections (private Galaxy)
- **Execution Environments** — Containerized Ansible execution
- **Event-Driven Ansible** — Reactive automation based on events
