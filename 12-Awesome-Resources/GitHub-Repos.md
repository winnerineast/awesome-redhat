# 📚 GitHub Repositories

> Curated list of GitHub repositories containing Ansible playbooks, S2I templates, and Kubernetes configurations.

---

## Code Repositories

- **[Ansible Examples](https://github.com/ansible/ansible-examples):** Official community playbooks demonstrating multi-tier application deployments.
- **[OpenShift GitOps Examples](https://github.com/redhat-developer-demos/openshift-gitops-examples):** Comprehensive repository for Argo CD sync tests, pipelines, and overlay folders.
- **[Software Collections (Sclorg)](https://github.com/sclorg):** The home of official S2I builder images for Python, Node.js, Ruby, PHP, and databases.
- **[Strimzi Kafka Operator](https://github.com/strimzi/strimzi-kafka-operator):** Upstream repository for running Kafka clusters on OpenShift.
