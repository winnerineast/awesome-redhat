# 📚 Interactive Hands-On Labs

> Curated platforms offering free, browser-based environments to practice RHEL system administration, OpenShift clustering, and Ansible automation.

---

## Free Lab Environments

- **[Red Hat Interactive Labs](https://lab.redhat.com/):** Free, browser-based environments to practice package management, storage configuration, container execution, and shell scripting.
- **[OpenShift Developer Sandbox](https://developers.redhat.com/developer-sandbox):** Provides 30 days of free access to a shared multi-tenant OpenShift cluster. Ideal for testing S2I builds, Helm deployments, and custom application configurations.
- **[Instruqt OpenShift Tracks](https://instruqt.com/public/topics/openshift):** Interactive tutorials covering Service Mesh, Serverless, GitOps, and ACM.
