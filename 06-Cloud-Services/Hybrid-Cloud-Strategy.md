# Hybrid Cloud Strategy
> Consistent application platform across on-premises, public cloud, and edge.

## Red Hat's Hybrid Cloud Stack
- **RHEL** — Consistent foundation
- **OpenShift** — Consistent application platform
- **ACM** — Unified multi-cluster management
- **Ansible** — Cross-environment automation
